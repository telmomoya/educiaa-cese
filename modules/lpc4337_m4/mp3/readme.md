Biblioteca ILI9341 SPI
==================

Biblioteca para el manejo del display TFT ILI9341 conectada por SPI.

Soporta fuentes de 7 x 10 y 11 x 18 pixels
Manejo de strings ( con \n y \n\r )
Dibuja rectángulos, círculos y líneas
Permite rotar el LCD
Display ON/OFF

Portada de una biblioteca escrita para STM32 por  [Tilen Majerle](https://stm32f4-discovery.net/tag/tilen-majerle/)
También usa el SPI de Matias Ferraro.


Conexión:
DC = GPIO0: Pin29
CS = GPIO1: Pin32
RESET = GPIO2: Pin31
MOSI= SPI_MOSI: Pin21
CLK= SPI_SCK: Pin20
